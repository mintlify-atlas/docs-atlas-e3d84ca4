## Template usage

```html
<!-- Minimal -->
<JInput :field="form.name" label="Name" />

<!-- Full options -->
<JInput
  :field="form.email"
  label="Email address"
  type="email"
  placeholder="you@example.com"
  required="true"
  helpText="We will never share your email."
  autocomplete="email"
  onInput="validate(form)"
/>
```

## Form binding pattern

`JInput` is designed to work alongside a server-side DTO annotated with Bean Validation. Set `field` to a dotted path that matches a field on your `@State` object. When a `@Call` method accepts that object, JReactive collects all `name="form.*"` values from the DOM and deserializes them into the parameter.

```java
@Route(path = "/signup")
@StatefulRam
@Client
public class SignupPage extends AppPage {

    public static class SignupForm implements Serializable {
        @NotBlank public String name;
        @NotBlank @Email public String email;
        @NotBlank @Size(min = 8) public String password;
    }

    @State SignupForm form = new SignupForm();
    @State String message = "";

    @Call
    public void register(@Valid SignupForm form) {
        message = "Welcome, " + form.name;
    }

    @Override
    protected String template() {
        return """
            <JForm onSubmit="register(form)">
              <JInput :field="form.name"  label="Name"     type="text"     required="true" />
              <JInput :field="form.email" label="Email"    type="email"    required="true" />
              <JInput :field="form.password" label="Password" type="password" required="true" />
            </JForm>
            <p>{{message}}</p>
            """;
    }
}
```
