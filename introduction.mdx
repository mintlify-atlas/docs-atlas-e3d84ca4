# JReactive

JReactive is a **Java-native reactive UI framework** that lets you build dynamic, live-updating web interfaces using pure Java — no frontend build tools, no npm, no TypeScript required. Components are plain Java classes that extend `HtmlComponent`. State is declared with annotations. The browser reacts to changes automatically via WebSocket-powered DOM patching.

## Why JReactive?

Modern reactive UIs typically require a JavaScript framework, a build pipeline, and a separate frontend codebase. JReactive eliminates that gap: your UI logic lives entirely in Java, co-located with your business logic, compiled by the JVM.

## Module overview

JReactive is structured as a multi-module Maven project:

| Module | Purpose |
|--------|---------|
| `jreactive-core` | Core abstractions: `HtmlComponent`, `ReactiveVar`, annotations, router SPI |
| `jreactive-runtime-jvm` | AST-based template engine and WebSocket protocol handler |
| `jreactive-runtime-js` | Browser-side runtime (WebSocket client, DOM morph, event proxy) |
| `jreactive-starter-spring` | Spring Boot auto-configuration and WebSocket integration |
| `jreactive-apt` | Annotation processor for AOT template and CSS compilation |
| `jreactive-ui` | Pre-built UI components (JButton, JInput, JForm, JTable, JModal, etc.) |
| `jreactive-store-redis` | Redis-backed distributed state store |
| `jreactive-standalone` | Standalone (non-Spring) bootstrap |
| `jreactive-demo-spring` | Full-featured demo application |

## Key annotations at a glance

```java
@Route(path = "/counter")              // Maps component to a URL
@Stateful                               // Serialize state to Redis
public class CounterPage extends HtmlComponent {

    @State int count = 0;              // Reactive state field

    @Call
    public void increment() {          // Callable from template events
        count++;
        updateState("count");
    }

    @Override
    protected String template() {
        return """
            <div>
              <p>Count: {{count}}</p>
              <button @click="increment()">+1</button>
            </div>
            """;
    }
}
```
